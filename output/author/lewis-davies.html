The Data Analysis Survival Guide

SQL & Python are incredibly powerful tools for analysing data, but there's a lot to learn about both. This site is my personal notebook on accessing, wrangling and analysing data, with reusable code snippets available as a reference.

This site is inspired by ChrisAlbon.com, which has been a huge help as I've learned to analyse data. This site is also open source and available on Github.
