Sorting Data

Import libraries

import numpy as np
import pandas as pd

Create DataFrame

data = {
    'name': ['Regina', 'Barbie', 'Gaz', 'Jiminy', 'Fran'],
    'number': np.random.randint(0, 100, 5)
}

df = pd.DataFrame(data)
df.head()
name number
0 Regina 40
1 Barbie 75

Sorting our values

# Sorting in alphabetical order
df.sort_values('name')

# Sorting numbers in reverse order
df.sort_values('number', ascending=False)
